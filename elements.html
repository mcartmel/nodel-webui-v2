New Nodel UI base layer. Navigation is generated automatically from declared pages. This page is a reference and proving ground for reusable Nodel components. Full width on small screens, half width from medium screens upward. This uses span="12" md="6" on each column. Core pages can be composed with plain HTML using the provided web components and stable v2 assets. Visual v2 pages are deployed as HTML files with JavaScript, CSS, and images under the v2 folder. The default Nodel document remains the non-visual index.htm redirector.
