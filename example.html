Collapse

Default collapsed section content. Section content shown after opening. Open section content. 
<nodel-row>
  <nodel-column md="4">
    <nodel-collapse label="Default">
      <nodel-text>Default collapsed section content.</nodel-text>
    </nodel-collapse>
  </nodel-column>
  <nodel-column md="4">
    <nodel-collapse label="With preview" preview="Preview shown while collapsed">
      <nodel-text>Section content shown after opening.</nodel-text>
    </nodel-collapse>
  </nodel-column>
  <nodel-column md="4">
    <nodel-collapse label="Open" preview="Starts open" open>
      <nodel-text>Open section content.</nodel-text>
    </nodel-collapse>
  </nodel-column>
</nodel-row>

  </nodel-column>

  <nodel-column span="12">

Text

Use nodel-text for themed text tones, sizes, and optional card surfaces.

Signal

Set signal to a local signal name to update the default target. For text, the default target is value; use signals for explicit bindings such as Status:value.

Waiting for Status signal 
<nodel-row>
  <nodel-column md="6">
    <nodel-text signal="Status" surface="card" tone="default">Waiting for Status signal</nodel-text>
  </nodel-column>
</nodel-row>

Visibility

Use visibility on any component to show or hide it from a local signal. Values visible, true, and 1 show the component; hidden, false, and 0 hide it.

Signal-controlled panel 
<nodel-row visibility="PanelVisible">
  <nodel-column md="6">
    <nodel-text surface="card" tone="default">Signal-controlled panel</nodel-text>
  </nodel-column>
</nodel-row>

Host Icon

Use nodel-host-icon to render a generated identicon. Its default signal target is host, with explicit bindings available for link and label attributes. 

<nodel-row>
  <nodel-column md="6">
    <nodel-host-icon host="Example Host" icon-host="example-host" title="Example host"></nodel-host-icon>
  </nodel-column>
  <nodel-column md="6">
    <nodel-host-icon signal="HostName" signals="HostAddress:icon-host; HostUrl:href; HostTitle:title" host="Waiting for host signal"></nodel-host-icon>
  </nodel-column>
</nodel-row>
